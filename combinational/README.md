# Combinational Circuits (Verilog)

## Overview

This directory contains implementations of fundamental combinational circuits using Verilog HDL.
These designs are built as part of a structured VLSI learning path, starting from basic logic gates and progressing toward complex digital systems.

---

## Included Modules

### Adders

* Half Adder
* Full Adder
* Ripple Carry Adder (RCA)
* Carry Lookahead Adder (CLA)

### Subtractors

* Half Subtractor
* Full Subtractor

### Multiplexers

* 2:1 MUX
* 4:1 MUX
* 8:1 MUX

### Comparator

* 4-bit Comparator

### ALU

* 2-bit Arithmetic Logic Unit

---

## Design Approach

* Implemented using Verilog HDL
* Focus on clarity, correctness, and modular design
* Includes testbenches and waveform verification for each module

---

## Tools Used

* Icarus Verilog (Simulation)
* GTKWave (Waveform Analysis)
* Vivado
---

## Learning Objective

* Understand combinational logic design
* Build strong foundation for sequential circuits and FSM design
* Develop structured and reusable hardware modules

---

## Note

Each subfolder contains:

* Design file (.v)
* Testbench
* Waveform output
* Documentation (README)

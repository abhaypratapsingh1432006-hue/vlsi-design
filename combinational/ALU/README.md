
# Arithmetic Logic Unit (ALU) Designs – Verilog

This folder contains my implementations of various Arithmetic Logic Units (ALUs) using Verilog HDL.  
The designs are developed as part of my learning in Digital VLSI and RTL Design.

---

##  Objective

- Understand ALU architecture and functionality
- Implement arithmetic and logic operations at RTL level
- Practice modular and scalable hardware design
- Verify designs using testbenches

---

##  Available Designs

###  2-bit ALU
- Supports basic arithmetic and logic operations
- Operations include: Addition, Subtraction, AND, OR
- Designed using combinational logic
- Verified using a self-checking testbench

---

##  Design Approach

- Behavioral / RTL modeling in Verilog
- Modular design using smaller building blocks
- Functional verification using simulation

---

##  Tools Used

- Verilog HDL  
- Xilinx Vivado (Simulation & Synthesis)  
- GTKWave (Waveform Analysis)  

---

##  Key Learnings

- ALU operation selection using control signals  
- Integration of arithmetic and logic units  
- Importance of verification in digital design  

---

##  Future Work

- Design 4-bit and 8-bit ALU  
- Add more operations (XOR, NOT, Shift operations)  
- Optimize design for speed and area  
- Explore pipelined ALU design  

---

##  Note

This folder will be continuously updated with more ALU designs and improvements as I progress in VLSI design.
